# RockBarefeetMod - 脱鞋 Mod

一款适用于 **洛克王国：世界** 游戏的脱鞋 Mod，让角色光脚行走。

---

## 使用方法

### 1. 下载注入器

前往 [RockDLL 注入器](https://github.com/Goldppx/RockDLL) 下载最新版本的注入器。

### 2. 注入 Mod

1. 编辑`config.ini`文件将路径设置为脱鞋Mod的文件路径
2. 双击启动`RockDLL.exe`注入器
3. 启动游戏

---

## 文件说明

| 文件 | 说明 |
|------|------|
| `DLL_lkwg_hijack.dll` | Mod 主体 DLL，注入到游戏进程 |

---

## 注意事项

- 穿初始鞋子为脱鞋效果，穿其他鞋子保持穿鞋效果
- 如遇游戏更新导致失效，请关注 QQ 群获取最新版本
